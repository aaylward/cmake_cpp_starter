# qwk
A simple project generator for c++ projects using CMake

### Usage
```bash
qwk new_thing
cd new_thing
code .
```

### Build Requirements
 - c++17 support in your compiler
 - Boost (for program_options)

### Build Instructions
```bash
git clone git@github.com:aaylward/qwk.git && cd qwk
cmake -Bbuild
cmake --build build
```
